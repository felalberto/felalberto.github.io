Teacher Resource 1

Teacher Overview

Use this overview to understand the project before introducing it to students.

CodeBronx
Code the Moment

What students will make

Students choose one small moment from a story and turn it into a short Scratch animation using:

  • characters
  • setting
  • dialogue
  • movement
  • sequence
  • debugging

Best classroom fit

Use this resource during:

  • narrative writing
  • character study
  • retelling
  • dialogue
  • sequence of events
  • point of view
  • creative response to text
CategoryRecommendation
Grade BandGrades 3 to 5. Adaptable for grades 2 to 6.
Time Needed
  • 45 minutes for a quick activity
  • 2 class periods for a mini-project
  • 3 class periods for a full build and share cycle
ToolScratch. Students may work individually or in pairs.
Final ArtifactA short animated scene that communicates one important story moment.
ELA Focus
  • character
  • setting
  • dialogue
  • action
  • sequence
  • theme
  • point of view
  • retelling
  • narrative development
CS/CT Focus
  • decomposition
  • sequencing
  • events
  • algorithms
  • debugging
  • optional loops or costume changes
Teacher reminder: Students should not code the entire story. A strong one-scene project is better than an unfinished whole-story project.
